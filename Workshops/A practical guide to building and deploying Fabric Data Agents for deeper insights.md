## Key Takeaways

Key community questions we captured and answered during/after the session (based on the latest Fabric updates in 2026):

Q: Will these integrate with partner solutions?

A: Yes — via OneLake interoperability, ISV ecosystem, and protocols like MCP for tools like Copilot Studio or custom apps.


Q: Does Data Agent work in a Fabric tenant with Private Endpoint? How to apply org DLP?

A: Yes — supports Azure Private Link/managed private endpoints. Purview DLP policies apply to Fabric items for sensitive data protection and compliance.


Q: Connecting a Data Agent to Copilot Agent — what’s needed for M365 Copilot?

A: Publish the agent in Fabric, enable tenant settings, and consume it directly in M365 Copilot for natural language queries over your data.


Q: Licensing beyond PBI Pro/PPU for workspace use?

A: Requires paid Fabric capacity (F2+) or Premium per capacity (P1+) with Fabric enabled. Pro/PPU alone isn't enough for creation/consumption.


Q: Plans for visuals in Data Agents? Convert Q&A to reports?

A: No native inline visuals yet (text-based focus), but use outputs to build Power BI reports manually. 
No auto-conversion feature announced.
